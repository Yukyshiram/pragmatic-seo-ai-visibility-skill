## How to use these

**`SKILL.md`** — A focused adaptation of the canonical skill for the specific site type. Use this as context when working on that type of site instead of the full canonical file, or alongside it.

**`audit-example.md`** — A realistic example of a full audit output (Mode B) for that site type. Use this to understand what a well-structured audit looks like in practice, or as a reference when producing your own audit outputs.
